# 🎥 YouTube Homepage Clone

A responsive YouTube homepage UI clone built using **HTML5** and **CSS3**. This project recreates the modern YouTube interface for learning and practicing frontend development concepts such as layouts, Flexbox, Grid, responsive design, and UI styling.

> ⚠️ This project is created for educational purposes only and is not affiliated with or endorsed by YouTube or Google.

## 🚀 Features

- Modern YouTube-inspired homepage
- Responsive design for desktop, tablet, and mobile
- Header with logo, search bar, and action icons
- Left sidebar navigation
- Video card grid layout
- Channel avatar and video information
- Hover effects and smooth transitions
- Clean and organized folder structure
- Semantic HTML5 markup
- Pure CSS3 (No frameworks)
